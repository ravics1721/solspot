# Solspot

## A solution sharing site

## **Geting Started**

## _*Client-app*_

#### `cd ss_client`

#### `npm i`

#### `npm start`

Run app in development mode.

## _*Server-app*_

#### `cd ss_server`

#### `npm i`

#### `npm run dev`

Run server in development mode .\
**_Note: Make Sure that you have `nodemon install globally` in your system_**
